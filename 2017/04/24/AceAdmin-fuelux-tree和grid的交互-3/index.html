J2EE

ace树添加点击事件

ace admin1.4.0一直没有出API，一般是从例子中来查询可用的方法,ace树自带的例子里并没有点击事件的操作，这样的话，只能从源码里查看对应的方法。源码的位置在ace_admin_v1.4.0\components\_mod\fuelux\tree.js。标准被JQuery插件的写法。
JavaScript

由于阅读源码的能力还是有些欠缺，对作者的一些想法把握不到位。所以在用起来这棵树的时候感觉有些吃力。本来一个简单功能实现起来并没有那么简单。

  • 1.点击事件场景为：点击folder节点，展开树的下一层，并在右侧grid显示该folder下层的子节点数 据。再点击收起树，右侧grid不刷新。
  • 2.selected事件中，有一段是调用tree.js中selectedItems的，这样写不太好。因为这棵树是单选的，这种写法适合多选树。
  • 这几行代码只是为了实现得到当前节点的ID，应该不用这么复杂，要找找更好的方法，从function函数中的参数中获得id值
  • 3.点击树后，右侧的grid进行同步的刷新，这里放弃了iframe的使用。而使用Jquery+div的形式来代替这样的场景。
  • 4.为了实现点击folder能展开能收起，还用了全局变量ab,这个ab就是存放当前el。这种写法应该也是可以改进的。这样写是因为在deselected找不到对应的el，应该是对源码的了解不够。
  • 5.最重要的一点是如果在构造树的时候，选择folderSelect : true的时候才会出现点击folder时展不开的问题。莫非作者提供了一个在选择folderSelect : true能展开树的属性我没有找到？这就尴尬了

tree selected事件简化

  • 1.这次简化主要是运用了事件中function的参数，从参数中获取值来进行操作
  • 2.全局变量ab还是去不掉，因为在deselected事件中，记不住当前取消的是哪个节点，也许有其他方法，后期再看看。
