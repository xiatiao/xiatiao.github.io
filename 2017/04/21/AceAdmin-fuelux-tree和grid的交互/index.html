AceAdmin fuelux tree和grid的交互(1)

J2EE
  • Ace Admin
  • fuelux tree

挺好看的一棵树，没有API

还是要强调一下，Ace Admin的版本为1.4.0
项目框架搭建好以后，应该是业务的开发，系统管理模块是必不可少的。具体业务也就不说了，这里先开发的是部门组织结构。在需求设计的时候，想了很长时间要不要把地域模块加进去，后来还是放弃了，直接上组织结构也能满足很多场景。

部门组织结构这块以前在HW的时候，对这块分析了较长的时间。界面上还是左树右表格的形式最为直观。下面来一张图

上图展示的就是部门树和部门表格之间的联动，这里的需求是(自己定的需求，毕竟SE嘛)。

  • 1.点击树上任何item，树节点和右边表格同时显示当前节点的下级菜单，
  • 2.点击右侧表格名称属性，树节点和右边表格同时显示当前节点的下级菜单
  • 3.点击表格区的返回按钮，树节点和右边表格同时显示当前节点的上级菜单(有点小bug，后期改)
